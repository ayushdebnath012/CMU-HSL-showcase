The map

One rigid transform, applied in splat space.

The output is not a 2D paste. The chair Gaussians are transformed into the real scene coordinate system and concatenated before rendering, so depth order and alpha compositing happen inside the original 3DGS rasterizer.

pscene = s R passet + t
  • Positionp' = s R p + t for every foreground Gaussian.
  • Scalelog-scale is offset by log(s) per axis.
  • Rotationasset quaternions are composed with the placement rotation and renormalized.
  • Appearanceopacity and DC color are blended toward local scene statistics with a hard opacity floor.

Field notes

Five failures found by rendering, not by wishing.

The math was compact. The implementation was not. The actual project became a debugging trail through config defaults, grounding, density schedules, floaters, baked color, and the late-discovered up-axis mismatch.

The chair that was almost invisible

Runner defaults were overriding the checked-in config, silently changing scale and opacity.

Floating above the grass

Placement height moved from manual guessing to scene point-cloud grounding plus camera calibration.

The confetti ball

The chair had reused the Bicycle VRAM-throttled densification schedule, capping it at about 48k Gaussians before retraining at about 289k.

The ghost halo

Floater Gaussians and white-background color were handled with four pruning filters and spherical-harmonic rest-band damping.

The up-axis trap

Blender's local up axis had to be aligned to the real scene's measured up vector using Rodrigues' rotation formula.

Extension

What shipped, and what is still an idea.

The current system includes a heuristic appearance match. The proposed extension is to make that correction learned and view-aware instead of one global statistical blend.

Shipped: statistical appearance match

The chair keeps 65% of its identity while blending 35% toward local scene color and brightness, with an opacity floor to prevent invisibility.

Proposed: learned appearance transfer

Optimize color correction against rendered feedback from multiple real cameras, rather than relying on a single averaged scene statistic.
